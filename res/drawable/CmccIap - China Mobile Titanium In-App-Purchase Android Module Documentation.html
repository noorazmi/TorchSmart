Introduction :

This module uses CMCC InappBilling SDK version 2.3.0 to utilize its services.

Accessing the ti.cmcc.iap Module

To access the module in javascript (app.js):

       var iap = require('ti.cmcc.iap');

Enabling/Disabling Logs

To enable/disable the logs of module in javascript (app.js):

       iap.setDebug(true/false);

By default the preference is set to false.

Exception Event

Use this event to get the exception logs of module in javascript (app.js):

       iap.addEventListener('exception', function(err){ 
       alert('[CMCC_IAP] Exception : \n' + err.errorMessage);
       });
